# Deep Playground（神经网络可视化）

Deep Playground 是一个交互式的神经网络可视化工具，使用 TypeScript 和 d3.js 编写。我们使用 GitHub issues 来跟踪新功能请求和错误报告。非常感谢您的反馈！

**如果您想贡献代码，请务必查看 [贡献指南](CONTRIBUTING.md)。**

## 致谢

特别感谢原项目作者 **Daniel Smilkov** 和 **Shan Carter**，以及 **Google** 团队开发了这个优秀的神经网络可视化工具！

这个项目是在 TensorFlow Playground 基础上进行的二次开发和改进。

## 主要修改说明

**修改人：DUUDU**
**修改日期：2026年**

本版本在原版 Deep Playground 基础上进行了以下改进：

### 1. 中文化
- 完全中文化的用户界面
- 翻译了所有说明文字、标签和提示信息
- 添加了中文使用说明和参数介绍

### 2. 新增功能
- **决策边界回放功能** - 可以回放神经网络的学习过程，展示决策边界的演变动画
- **隐藏层激活可视化** - 通过颜色展示每层神经元的激活状态
- **更多数据集** - 新增了多种分类和回归数据集，包括：
  - 棋盘格、月牙、心形、对角线、环形、波浪、三角形等分类数据集
  - 正弦波、碗形、山丘、瑞士卷、径向、棋盘格回归、马鞍形等回归数据集
- **更多输入特征** - 特征从原来的7个增加到13个，包括：
  - cos(X₁)、cos(X₂) - 余弦函数
  - |X₁|、|X₂| - 绝对值
  - X₁³、X₂³ - 立方
  - exp(X₁)、exp(X₂) - 指数函数（归一化）
  - σ(X₁)、σ(X₂) - Sigmoid激活函数

### 3. 界面优化
- 重新组织了控制组件的布局，使操作更方便
- 改进了响应式设计
- 优化了视觉效果和用户体验

### 4. 文档完善
- 新增了详细的中文文档
- 提供了程序工作原理说明
- 添加了使用方法和参数指南
- 给出了获得最佳结果的建议

## 开发说明

要在本地运行可视化程序，请执行以下步骤：
- `npm i` 安装依赖
- `npm run build` 编译应用并将其输出到 `dist/` 目录
- `npm run serve` 从 `dist/` 目录提供服务并在浏览器中打开页面

开发时为了获得快速的编辑-刷新循环，请运行 `npm run serve-watch`。
这会启动一个 http 服务器，并在 TypeScript、HTML 和 CSS 文件更改时自动重新编译它们。
